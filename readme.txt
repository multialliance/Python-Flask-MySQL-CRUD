# CRUD Python-Flask-MySQL

A simple CRUD application with Python, Flask framework and MySQL DB

## Description

This is a simple project based in Python, Flask and MySQL, that allows you to create, consult,
edit and erase users, using email as username, name and password.

## Getting Started

### Dependencies

* First you have to create a MySQL database with a table named users and id, username, name and password
* Make the libraries installation (Flask, mysql-connector-python, os)
* Then you have to make the connection to the DB, filling the informations required in database.py file.

### Installing

* Create virtual environment and use requirements.txt to install the libraries.
* Any modifications needed to be made to files/folders
